- Webpack
- JavaScript-CS6

## 🤖 Getting started

- In the desired directory, type `git clone https://github.com/abmAsadullah/js-todo.git` then `cd js-todo` in the terminal.

- Run `npm install`.

- Run `npm run server`.

- See your Brawser `http://localhost:3000/`.
